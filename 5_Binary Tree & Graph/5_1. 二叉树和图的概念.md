# 1. 二叉树和图的概念

# 1.1. 二叉树

二叉树的数据结构
```c++
struct TreeNode{
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x): val(x),left(NULL),right(NULL){}
};
```

二叉树的遍历
```
              5
             / \
            4   8
           /   / \
          11  13  4
         /  \    / \
        7    2  5   1
```

前、中、后序遍历，又称为深度优秀搜索(深搜)

前序遍历：5-4-11-7-2-8-13-4-5-1

中序遍历：7-11-2-4-5-13-8-5-4-1

后序遍历：7-2-11-4-13-5-1-4-8-5


```c++
void traversal(TreeNode *node){
    if (!node){
        return;
    }
    //此时访问node, 称为 【前序遍历】
    traversal(node->left);
    //此时访问node, 称为 【中序遍历】
    traversal(node->right);
    //此时访问node, 称为 【后序遍历】
}
```

层次遍历，又称为宽度优秀搜索(宽搜)

上述二叉树的层次遍历：5-4-8-11-13-4-7-2-5-1

用队列这种数据结构来实现二叉树的宽搜

```c++
#include <queue>
void BFS(TreeNode* root){
    std::queue<TreeNode*> Q;
    Q.push(root);
    while(!Q.empty()){
        TreeNode *node = Q.front();
        Q.pop();
        printf("[%d]\n", node->val);
        if (node->left){
            Q.push(node->left);
        }
        if (node->right){
            Q.push(node->right);
        }
    }
}
```
